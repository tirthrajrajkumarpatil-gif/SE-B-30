💰 Smart Expense Manager

📌 Overview

Smart Expense Manager is a simple web-based application that helps users track their daily expenses. Users can add, delete, and filter expenses, while also viewing the total amount spent.

⸻

🎯 Features

* ➕ Add expenses (name, amount, category)
* 🗑️ Delete expenses
* 🔍 Filter expenses by category (Food, Travel, Shopping, Other)
* 💾 Data stored in browser using LocalStorage
* 📊 Automatic total expense calculation

⸻

🛠️ Technologies Used

* HTML – For structure
* CSS – For styling and UI
* JavaScript – For logic and functionality
* LocalStorage – For saving data permanently in browser
* ⚙️ How It Works

1. User enters expense name, amount, and category
2. Clicks on “Add” button
3. Expense gets added to the list
4. Data is saved in LocalStorage (persists after refresh)
5. User can filter expenses using dropdown
6. Total amount updates automatically
7. User can delete any expense using delete button
